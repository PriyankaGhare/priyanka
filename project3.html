Vehicle Pooling Application

This project presents the development of a mobile-based vehicle pooling application aimed at reducing travel costs, traffic congestion, and environmental pollution.

The application facilitates ride-sharing by enabling users and vehicle owners to register, log in, and coordinate journeys through a user-friendly interface. Users can check meeting points, view real-time locations of drivers, and schedule rides.

By automating the process of finding and joining vehicle pools, the application simplifies the logistics of ride-sharing, making it a convenient, eco-friendly, and cost-effective travel solution.

Additionally, the application includes a notification system to keep all participants updated on journey details and changes, further enhancing the reliability and efficiency of the ride-sharing experience.

Technologies Used:
JAVA
Firebase
RESTful APIs
GeoFence API
Docker
Agile
